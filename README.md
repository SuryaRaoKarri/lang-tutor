# Language Auto-Loop Tutor

A web-based tool for language learners to practice French vocabulary using text-to-speech loops. 

## Features
- **Auto-Looping:** Set your own repetition counts for vocabulary.
- **Customizable:** Adjust speech speed, pause timings, and select from system voices.
- **Mobile Friendly:** Works directly in your mobile browser.
- **No Backend:** Runs entirely in your browser using the Web Speech API.

## How to use
1. Paste your vocabulary list (format: `French - English`) into the text area.
2. Click "Load List".
3. Adjust your speed and pause settings.
4. Click "Play All" to start your practice session.
